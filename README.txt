*THIS PROGRAM GENERATES AN ARBITRARY ARRAY AND FINDS THE
DIFFERENCE BETWEEN THE MAXIMUM AND MINIMUM ELEMENTS OF EACH ROW*

1. Open "Array generator"
2. Enter field value in field "Рядки"
3. Enter row value in field "Стовпці"
4. Click "Згенерувати масив"
5. Change items or not
6. Click "Результат"
7. Done
